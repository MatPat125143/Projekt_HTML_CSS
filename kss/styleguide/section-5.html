5.1 #permalink Gallery Container

Kod HTML

  • <section class="gallery"> - Sekcja dla galeri zdjęć.
  • <div class="column" id="column1"> - Kotener dla pierwszej kolumny zdjęć.

Kod CSS

  • .gallery - Ustawienie pozycji, kolorów i wysokości dla sekcji gallery.
  • .column - Ustawienia kolumn zdjęć.
  • .column img - Ustawienie wyświetlanie i pozycji dla zdjęć.
  • #overlay, 1#overlay img` - Ustawienie wyświetlania poglądu zdjęcia po jego naciśnięciu.

Kod JS

  • function showImage(img) - Wyświetlenie zdjęcia na podglądzie.
  • function hideImage() - Schowanie zdjęcia na podglądzie.

5.3 #permalink Gallery Conainer Script

Toggle full screen Open in new window Toggle example guides Toggle HTML markup 

<script>
    function showImage(img) {
        var overlay = document.getElementById('overlay');
        var overlayImage = document.getElementById('overlayImage');

        overlayImage.src = img.src;
        overlay.style.display = 'flex';
    }
    function hideImage() {
        var overlay = document.getElementById('overlay');
        overlay.style.display = 'none';
    }
</script>
