2.1 #permalink Navigation Bar

Panel nawigacyjny dla wszystkich stron.

Kod HTML

  • <nav class="navbar" id="topnav"> - Zawiera logo strony oraz wszystkie łącza nawigacyjne.
  • <div class="logo"> - Kontener przechowujący logo.
  • <ul class="nav-links"> - Lista nieuporządkowana z łączami nawigacji.
  • <label for="checkbox_toggle" class="hamburger"> - Chcekbox odpowiedzialny za rozwijanie menu (hamburger menu).
  • <nav class="menu"> - Kontener zawierający łącza nawigacyjne.
  • <li><a href="\"> - Łącznik nawigacyjny do danej podstrony (przykład).

Kod CSS

  • .navbar - Ustawienia kontenera elastycznego, wyśrodkowanie go w pionie oraz przyczepienie go na stałe do góry. Nadaje również kolor oraz cień całego menu.
  • .nav-links - Ustawienie stylu listy.
  • .nav-links a - Ustawienie koloru, dekoracji oraz rozmiaru teksu dla łącz nawigacji.
  • .logo oraz .logo a - Ustawienia sposobu wyświetlania, wysokości, koloru oraz cienia adla loga.
  • .logo a:hover - Animacja zmiany koloru po najechaniu na logo.
  • .menu, .menu li, .menu li;hover - Nadanie łączom koloru, rozmiaru oraz animacji po najechaniu.
  • input[type=checkbox] - Ustawienie checkboxa hamburgera na niewidoczny.
  • .hamburger - Określenie wyświetlania hamburgera, wielkości i cienia.

2.3 #permalink Footer

Stopka dla wszystkich stron. Kod HTML

  • <footer> - Stopka stro.
  • <section class="social-bar"> - Sekcja przechowująca icony do social mediów.
  • <a href="" class="icons-social"> - Linki z zdjęciami social mediów.

Kod CSS

  • footer - Ustawienie pozycji, koloru, szerokości oraz wyśrodkowanie zawartośći.
  • .social-bar, .social-bar h4, .icons-social - Nadanie koloru teksu, ustawienie marginesów.
